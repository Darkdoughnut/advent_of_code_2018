# # Advent of code 2018

My run of this [event](https://adventofcode.com/2018) in Rust. 
Shoutout to [porglezomp](https://gist.github.com/porglezomp).

## Getting Started

Requires an installation of rust.

### To build
```
cargo build
```

### To test
```
cargo test
```
